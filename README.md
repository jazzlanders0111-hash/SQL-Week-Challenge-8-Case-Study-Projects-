## 👋 About This Repository

This repository contains my solutions to all 8 case studies from the [#8WeekSQLChallenge](https://8weeksqlchallenge.com/) — a challenge that pushed me to apply SQL in real-world business scenarios and sharpen my query writing and analytical thinking along the way.

Huge thanks to **Data with Danny** for building something this genuinely useful for people learning data. 🤝

---

## 💡 How I Found This

I came across **Katie Huang's** repository ([katiehuangx/8-Week-SQL-Challenge](https://github.com/katiehuangx/8-Week-SQL-Challenge/tree/main)) while looking for SQL projects to practice with.

Her work inspired me to take on the challenge myself — but I want to be transparent: **I did not copy her solutions.** I started from scratch, read each question carefully, broke down the logic on my own, and wrote every query myself. Her repo showed me what was possible. The work here is my own.

---

## 📌 What This Is

The [#8WeekSQLChallenge](https://8weeksqlchallenge.com/) is a free SQL challenge created by [Danny Ma](https://www.linkedin.com/in/datawithdanny/) — 8 case studies, each based on a real-world business scenario. No hand-holding, no multiple choice. Just you, the data, and SQL.

## 🛠️ Tools Used

- **PostgreSQL** — all queries written and tested here

---

## 🧠 Concepts Covered

- SELECT, WHERE, ORDER BY, LIMIT
- Aggregate Functions — SUM, COUNT, AVG, MIN, MAX
- GROUP BY and HAVING
- JOINs — INNER, LEFT, RIGHT, FULL, CROSS
- CTEs (Common Table Expressions)
- Window Functions — DENSE_RANK, ROW_NUMBER, LAG, LEAD
- CASE WHEN conditional logic
- Date Functions and Filtering
